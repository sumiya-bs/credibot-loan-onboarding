# CrediBot — Conversational AI That Transforms Loan Onboarding from Days to Seconds

## What is CrediBot?
An AI-powered loan onboarding assistant built on n8n that handles 
eligibility checks, EMI calculations, document verification, loan 
application filing, and status tracking — all through a simple 
conversational chat interface.

## Tech Stack
- n8n (Workflow Automation)
- OpenAI GPT (AI Agent & Chat Model)
- Google Sheets (Real-time Database)

## Workflow
Import `CREDIBOT_ONBOARDING_ASSISTANT.json` directly into your n8n instance.

## Features
- Loan Eligibility Check
- EMI Calculator
- Document Verification
- Loan Application Filing
- Real-time Status Tracking
